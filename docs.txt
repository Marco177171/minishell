readline(const char *prompt)
    It returns the text contained in *prompt. We will use it to return recursively the text "minishell$ " at the begin of the program and at the end of each interaction.
    It is contained in the library <readline/readline.h>.

rl_clear_history
    It deletes the history when the user quits the program using the "exit" command.
    It is contained in the library <readline/readline.h>
